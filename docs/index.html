Senzing Free Support

Senzing offers free support for paid, complimentary, and trial licenses.

Community forums

Senzing community discussions are located at https://senzing.community. In the discussions you can learn how to integrate SenzingAPI and Senzing Community resources into your existing infrastructure.

GitHub issues

To request features or report bugs in individual repositories, visit the repository of interest in the Senzing community, click on "issues", then click on "New issue" to open a new request.

If a new feature that spans repositories or requires a new repository is requested, first see if it is already on the the Senzing Community Roadmap. If it isn't on the roadmap, make a new request.

Email

If you want to keep private information out of public discussion groups, feel free to send email to support@senzing.com.
